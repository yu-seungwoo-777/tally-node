## 1. RX 모드 개요

### RX 모드란?

RX (Receiver) 모드는 TX 장치에서 LoRa 무선으로 Tally 정보를 수신하여 LED와 디스플레이로 카메라맨에게 알려주는 휴대용 장치입니다.

**주요 기능:**
- LoRa 무선 Tally 수신 (거리: 최대 500m)
- LED 색상 표시 (Program/Preview)
- 디스플레이 정보 표시 (카메라 ID, RSSI, 배터리)
- 배터리 구동 (최대 8시간)

### 사용 시나리오

| 시나리오 | 설명 |
|----------|------|
| **스튜디오 촬영** | 카메라맨이 자신의 Tally 상태를 실시간 확인 |
| **외부 촬영** | 이동이 자유로운 무선 Tally 표시 |
| **다중 카메라** | 최대 20개의 서로 다른 카메라 ID 지원 |

### 필수 하드웨어

- [x] Tally Node RX 장치
- [x] 리튬이온 배터리 (3.7V, 18650)
- [x] 배터리 충전기 (Type-C 포트)

## 2. 하드웨어 설치

### 2.1 배터리 설치

> **경고**: 반드시 보호 회로가 있는 18650 리튬이온 배터리를 사용하세요. 저가형 보호 없는 배터리는 화재/폭발 위험이 있습니다.

**단계:**

1. RX 장치의 뒷면 커버를 엽니다
2. 배터리 홀더를 확인합니다 (+, - 극성 주의)
3. 배터리의 **양극(+)이 위쪽**을 향하도록 삽입합니다
4. 배터리가 확실히 고정되는지 확인합니다
5. 뒷면 커버를 닫습니다

**예상 결과:**
- 배터리가 제자리에 고정됩니다
- 흔들어도 움직이지 않습니다

### 2.2 카메라 장착

RX 장치를 카메라에 장착하는 방법:

#### 옵션 A: 핫슈 마운트

1. 카메라의 핫슈 슬롯을 확인합니다
2. RX 장치 하단의 핫슈 마운트를 슬롯에 끼웁니다
3. 고정 나사를 조여 장치를 고정합니다

#### 옵션 B: 벨트 클립

1. 벨트 클립을 RX 장치에 부착합니다
2. 카메라맨의 벨트나 옷에 클립을 고정합니다
3. 장치가 흔들리지 않는지 확인합니다

#### 옵션 C: 매직 테이프

1. 매직 테이프(벨크로)를 준비합니다
2. 한 조각은 카메라에, 다른 조각은 RX 장치에 부착합니다
3. 두 조각을 붙여 장치를 고정합니다

### 2.3 LED/디스플레이 위치

장치 전면에는 다음 표시등이 있습니다:

| 표시 | 위치 | 설명 |
|------|------|------|
| **Program LED** | 상단 | 적색: 방송 중 |
| **Preview LED** | 중단 | 녹색: 예비 됨 |
| **디스플레이** | 하단 | 카메라 ID, 신호 강도, 배터리 |

## 3. 전원 켜기

### 3.1 전원 버튼 조작

**전원 켜기:**

1. 측면 전원 버튼을 **2초간 길게 누릅니다**
2. 부팅 화면이 나타납니다
3. 디스플레이 로고가 표시됩니다

**전원 끄기:**

1. 전원 버튼을 **2초간 길게 누릅니다**
2. "전원 끄기..." 메시지가 표시됩니다
3. 화면이 꺼집니다

### 3.2 부팅 화면 설명

부팅 시 다음 순서로 화면이 표시됩니다:

```
1. Tally Node 로고 (2초)
   ┌─────────────────┐
   │  TALLY NODE     │
   │    RX Mode      │
   └─────────────────┘

2. 초기화 중 (3초)
   ┌─────────────────┐
   │  LoRa 초기화... │
   │  디스플레이...   │
   └─────────────────┘

3. 대기 화면
   ┌─────────────────┐
   │ CAM: 01         │
   │ RSSI: --        │
   │ BAT: 100%       │
   └─────────────────┘
```

### 3.3 첫 부팅 초기 설정

처음 켜는 경우 다음 설정 단계가 자동으로 시작됩니다:

**단계:**

1. 언어 선택이 표시됩니다 (한국어/English)
2. 전원 버튼을 눌러 선택합니다
3. 카메라 ID 설정 화면으로 이동합니다 (다음 섹션 참조)

## 4. 카메라 ID 설정

### 4.1 ID 개요 (1-20)

Tally Node는 **1부터 20까지의 카메라 ID**를 지원합니다.

각 카메라맨은 고유한 ID를 할당받습니다:
- 카메라 1번: ID = 1
- 카메라 2번: ID = 2
- ...
- 카메라 20번: ID = 20

**ID와 TX 장치 매핑:**
TX 장치에서 각 카메라에 해당하는 Tally 정보를 전송하면, 해당 ID를 가진 RX 장치가 수신합니다.

### 4.2 ID 설정 방법

**단계:**

1. 전원이 켜진 상태에서 전원 버튼을 **짧게 눌러서** ID 설정 모드로 진입합니다
2. 디스플레이에 현재 ID가 표시됩니다 (초기값: 1)
3. 전원 버튼을 누를 때마다 ID가 1씩 증가합니다 (1 → 2 → 3 → ... → 20 → 1)
4. 원하는 ID가 표시되면 **5초간 대기**합니다
5. "ID 저장 완료" 메시지가 표시됩니다

**예시: 카메라 3번 설정**

```
1. 버튼 누름: "CAM: 01"
2. 버튼 누름: "CAM: 02"
3. 버튼 누름: "CAM: 03"  ← 원하는 ID
4. 5초 대기
5. "저장 완료!"
```

### 4.3 ID 변경 방법

이미 설정된 ID를 변경하려면:

1. 전원을 켭니다
2. 전원 버튼을 **3초간 길게 눌러서** ID 설정 모드로 진입합니다
3. 위의 [ID 설정 방법](#42-id-설정-방법)을 따릅니다

> **팁**: 카메라맨끼리 ID가 중복되지 않도록 주의하세요. 중복 시 두 장치가 동일한 Tally를 수신합니다.

## 5. Tally 표시 이해

### 5.1 LED 색상 의미

| 상태 | Program LED | Preview LED | 의미 |
|------|-------------|-------------|------|
| **Safe** | 꺼짐 | 꺼짐 | 방송 안 함, 카메라 자유 |
| **Preview** | 꺼짐 | 🟢 녹색 | 다음 순서 예비, 준비 필요 |
| **Program** | 🔴 적색 | 꺼짐 | **지금 방송 중**, 조용히 |
| **Both** | 🔴 적색 | 🟢 녹색 | 방송 중 + 예비 (드물음) |

### 5.2 디스플레이 정보 해석

**대기 화면 (Safe 상태):**

```
┌─────────────────┐
│ CAM: 03         │ ← 카메라 ID
│ RSSI: -65 dBm   │ ← 신호 강도
│ BAT: 85%        │ ← 배터리 잔량
└─────────────────┘
```

**Tally 수신 화면 (Program 상태):**

```
┌─────────────────┐
│ ● PROGRAM      │ ← 방송 중 표시
│ CAM: 03         │
│ RSSI: -65 dBm   │
│ BAT: 85%        │
└─────────────────┘
```

**예비 상태 화면 (Preview 상태):**

```
┌─────────────────┐
│ ○ PREVIEW      │ ← 예비 표시
│ CAM: 03         │
│ RSSI: -65 dBm   │
│ BAT: 85%        │
└─────────────────┘
```

### 5.3 RSSI/SNR 표시

**RSSI (수신 신호 강도 지표):**

| RSSI 값 | 신호 상태 | 거리 추정 |
|---------|-----------|-----------|
| **-50 ~ -60 dBm** | 우수 | 50m 이내 |
| **-60 ~ -70 dBm** | 양호 | 50~150m |
| **-70 ~ -85 dBm** | 보통 | 150~300m |
| **-85 ~ -100 dBm** | 약함 | 300~500m |
| **< -100 dBm** | 매우 약함 | 통신 불안 |

**SNR (신호대잡음비):**
- **10 dB 이상**: 안정적인 통신
- **5 ~ 10 dB**: 간헐적 통신
- **5 dB 미만**: 통신 불안

> **팁**: RSSI가 -85 dBm 미만이면 TX 장치를 가까이 이동하거나 송신 전력을 높이세요.

## 6. 배터리 관리

### 6.1 배터리 잔량 확인

디스플레이 항상 배터리 잔량이 표시됩니다:

```
BAT: 100% ← 완충
BAT: 75%  ← 양호
BAT: 50%  ← 사용 중
BAT: 25%  ← 경고
BAT: 10%  ← 충전 필요
```

**LED 배터리 표시:**

| 상태 | LED 동작 | 의미 |
|------|----------|------|
| **정상** | 정상 표시 | 배터리 25% 이상 |
| **경고** | 적색 2초 간격 깜빡임 | 배터리 25% 미만 |
| **비상** | 적색 1초 간격 깜빡임 | 배터리 10% 미만 |

### 6.2 충전 방법

**충전 단계:**

1. RX 장치 전원을 끕니다 (권장) 또는 켜진 상태로 둡니다
2. Type-C 충전 케이블을 장치의 충전 포트에 연결합니다
3. 케이블另一端을 USB 충전기 또는 컴퓨터에 연결합니다
4. 충전 LED가 주황색으로 켜지는지 확인합니다

**충전 LED 상태:**

| 상태 | LED 색상 | 의미 |
|------|----------|------|
| **충전 중** | 🟠 주황색 (깜빡임) | 충전 진행 중 |
| **완충** | 🟢 녹색 | 충전 완료 |
| **충전 안 됨** | 🔴 적색 | 충전 오류 |

**충전 시간:**

| 배터리 상태 | 충전 시간 |
|-------------|-----------|
| **0 → 100%** | 약 3시간 |
| **50 → 100%** | 약 1.5시간 |
| **빠른 충전(30분)** | 0 → 30% |

### 6.3 배터리 경고 대응

**배터리 25% 경고:**

1. 현재 촬영이 곧 끝난다면 계속 진행합니다
2. 여유 시간이 있다면 예비 배터리로 교체합니다
3. 충전 중에도 사용 가능하지만 충전 속도가 느려집니다

**배터리 10% 비상:**

1. 가능한 빨리 배터리를 교체합니다
2. 촬영 중이라서 교체할 수 없는 경우:
   - 디스플레이 밝기를 낮춥니다 (설정 메뉴)
   - 불필요한 기능을 끕니다
3. 촬영 중지가 불가피하다면 PD(감독)에게 미리 알립니다

**배터리 교체 단계:**

1. 전원을 끕니다
2. 뒷면 커버를 엽니다
3. 배터리를 제거합니다
4. 새 배터리를 (+극 위쪽) 삽입합니다
5. 커버를 닫습니다
6. 전원을 켭니다

## 7. 일반 사용

### 7.1 카메라맨 작업 흐름

**일반적인 라이브 방송 워크플로우:**

1. **촬영 전 준비 (15분 전)**
   - 배터리 충전 확인 (최소 75%)
   - 전원 켜기 및 ID 확인
   - TX 장치와 통신 확인 (RSSI 체크)

2. **리허설**
   - Safe 상태에서 대기 (LED 꺼짐)
   - Preview 상태 시 녹색 LED 확인 → 자세 조정
   - Program 상태 시 적색 LED 확인 → 조용히 유지

3. **본 방송**
   - 내 Tally가 적색(Program)이면 움직이지 않습니다
   - 내 Tally가 녹색(Preview)이면 다음 순서 준비합니다
   - 내 Tally가 꺼져 있으면(Safe) 자유롭게 움직입니다

4. **방송 종료 후**
   - 전원 끄기
   - 배터리 충전
   - 다음 촬영까지 보관

### 7.2 장비 부착 위치

**권장 장착 위치:**

| 위치 | 장점 | 단점 |
|------|------|------|
| **카메라 핫슈** | 카메라맨의 시야 내 항상 표시 | 카메라 무게 증가 |
| **모니터 옆** | 촬영 중 보기 편함 | 별도 장착 필요 |
| **벨트/클립** | 무게 부담 없음 | 확인하려면 고개를 숙여야 함 |
| **삼각대 다리** | 항상 고정된 위치 | 거리가 멀어서 잘 안 보일 수 있음 |

**최적 위치:** 카메라 측면 또는 모니터 옆 (카메라맨의 자연스러운 시야)

### 7.3 주의 사항

**사용 중 주의:**

> **경고**: 다음 상황에서는 주의가 필요합니다

1. **장비 운반 시**
   - 배터리가 빠지지 않도록 커버를 확실히 닫으세요
   - 낙하 방지를 위해 목걸이 또는 스트랩 사용을 권장합니다

2. **야외 촬영 시**
   - 방수가 되지 않습니다. 비나 눈에 노출되지 마세요
   - 직사광선 노출 시 디스플레이가 잘 안 보일 수 있습니다

3. **통신 거리**
   - TX 장치와의 거리가 300m 이내인지 확인하세요
   - 콘크리트 벽이나 금속 구조물은 신호를 차단합니다

4. **배터리 보관**
   - 장기간 미사용 시 배터리를 50% 충전 상태로 보관하세요
   - 3개월마다 한 번씩 충전하여 수명을 유지하세요
