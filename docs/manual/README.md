## Tally Node 개요

Tally Node는 LoRa 무선 통신을 기반으로 한 무선 Tally 시스템입니다.

### 주요 기능

- **TX 모드**: 비디오 스위처(ATEM, vMix, OBS)와 연동하여 Tally 정보 수집 및 LoRa 송신
- **RX 모드**: LoRa로 Tally 정보를 수신하여 LED/디스플레이로 카메라맨에게 표시
- **듀얼모드**: 두 대의 스위처 동시 연결 및 장애 자동 전환
- **웹 UI**: 브라우저 기반 설정 및 모니터링

### 시스템 구조

```
┌─────────────┐                    ┌─────────────┐
│  TX 장치     │                    │  RX 장치     │
│             │                    │             │
│  ┌───────┐  │    LoRa 무선       │  ┌───────┐  │
│  │ 스위처 │  │  ──────────────>  │  │ LED   │  │
│  │ 연동   │  │   923 MHz         │  │ 표시   │  │
│  └───────┘  │                    │  └───────┘  │
└─────────────┘                    └─────────────┘
     (1대)                            (최대 20대)
```

## 빠른 참조

### TX 모드 기본 설정

1. 전원 및 네트워크 연결
2. 웹 UI 접속: `http://tallynode-tx.local`
3. 스위처 연결 설정 (ATEM/vMix)
4. LoRa 파라미터 확인 (주파수: 923 MHz)
5. Tally 송신 확인

### RX 모드 기본 설정

1. 배터리 장착
2. 전원 켜기 (전원 버튼 2초)
3. 카메라 ID 설정 (버튼으로 선택)
4. Tally 수신 확인 (LED/디스플레이)
