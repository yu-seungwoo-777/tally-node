## 4. 업그레이드 가이드

### v1.0.x → v1.1.x

**새로운 기능**:
- 듀얼모드 (Primary/Secondary 스위처)
- RX 장치 상태 관리

**업그레이드 전 확인사항**:
- 기존 NVS 설정은 그대로 사용 가능
- Secondary 스위처 설정은 추가 설정 필요

### v1.1.x → v1.2.x

**새로운 기능**:
- 웹 UI 개선 (Alpine.js + TailwindCSS)
- SSE 기반 실시간 업데이트
- 다국어 지원

**업그레이드 전 확인사항**:
- 웹 UI 캐시 삭제 필요
- 브라우저 새로고침 권장

### v1.2.x → v1.3.x

**새로운 기능**:
- RX 모드 우선순위 최적화
- 배터리 Empty 페이지
- Deep Sleep 카운트다운

**업그레이드 전 확인사항**:
- NVS 마이그레이션 권장
- 배터리 임계값 설정 확인

### v1.3.x → v1.4.x

**새로운 기능**:
- TX 모드 Tally UI
- 라이선스 키 보호 기능

**업그레이드 전 확인사항**:
- 라이선스 키 백업 권장

### v1.x.x → v2.0.0

**새로운 기능**:
- OTA 업데이트
- HTTPS 지원
- OBS 프로토콜 정식 지원

**업그레이드 전 확인사항**:
- 주요 버전 업데이트로 NVS 재설정 권장
- 모든 설정 백업 필수
