# Tally Node 기능 카탈로그

## 개요

Tally Node의 모든 하드웨어, 소프트웨어, 프로토콜, 구성 옵션을 포괄하는 기능 카탈로그입니다.

## 카탈로그 구조

| 문서 | 설명 |
|------|------|
| [hardware.md](hardware.md) | 하드웨어 사양 및 구성 요소 |
| [software-features.md](software-features.md) | 소프트웨어 기능 및 모듈 |
| [protocols.md](protocols.md) | 스위처 프로토콜 지원 현황 |
| [configuration.md](configuration.md) | 구성 옵션 및 설정값 |
| [version-matrix.md](version-matrix.md) | 버전별 기능 매트릭스 |

## 상태 표시기

| 표시 | 의미 |
|------|------|
| ✅ 지원 | 현재 안정적으로 지원되는 기능 |
| 🔄 개발 중 | 개발이 진행 중인 기능 |
| 📋 계획 | 개발이 계획된 기능 |
| ❌ 폐지 | 더 이상 지원되지 않는 기능 |

## 카탈로그 사용 가이드

### 타겟 독자

- **시스템 통합자**: Tally Node를 자사 시스템에 통합하려는 엔지니어
- **방송 엔지니어**: 방송 시스템을 설계 및 구축하는 담당자
- **기술 지원 personnel**: 고객 지원 및 문제 해결 담당자
- **영업 팀**: 제품 기능을 고객에게 소개하는 담당자

### 문서 검색 팁

1. **빠른 검색**: 문서 내 검색(Ctrl+F)을 사용하여 원하는 기능을 찾으세요
2. **관련 문서**: 각 문서 하단의 관련 링크를 참조하세요
3. **버전 확인**: [version-matrix.md](version-matrix.md)에서 원하는 버전의 기능 지원 여부를 확인하세요

### 기능 상태 확인

- **✅ 지원**: 바로 사용할 수 있는 기능
- **🔄 개발 중**: 다음 릴리스에 포함될 예정인 기능
- **📋 계획**: 로드맵에 포함된 기능
- **❌ 폐지**: 레거시 지원만 제공되는 기능

## 기술 지원

카탈로그에 없는 기능이나 명확하지 않은 사항은 프로젝트 저장소의 Issue를 통해 문의해 주세요.
