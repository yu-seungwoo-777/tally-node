# 디스플레이 아이콘 구현

## 개요
RX/TX 페이지에서 사용하는 상태 아이콘 (배터리, 안테나 신호)을 구현했습니다.

## 구현된 기능

### 1. 배터리 아이콘
- 5단계 표시: 100%, 75%, 50%, 25%, 저전력(10% 미만)
- 저전력 상태는 반전되어 표시되어 경고 효과 제공
- 위치: RX/TX 페이지 왼쪽 상단 (x=5, y=6)

### 2. 안테나 신호 아이콘
- 4단계 표시: 강함, 중간, 약함, 없음
- RSSI 값에 따라 자동 선택
  - 강함: > -50dBm
  - 중간: -50 ~ -70dBm
  - 약함: -70 ~ -85dBm
  - 없음: < -85dBm
- 위치: RX/TX 페이지 오른쪽 상단 (x=107, y=6)

## 사용 예제

### 배터리 아이콘 그리기
```c
#include "icon_renderer.h"

// U8g2 인스턴스 가져오기
u8g2_t* u8g2 = DisplayHelper_getU8g2();

// 배터리 백분율에 따라 자동으로 아이콘 선택
icon_draw_battery_u8g2(u8g2, 85, 5, 6);  // 85%, x=5, y=6
```

### 신호 강도 아이콘 그리기
```c
// RSSI 값에 따라 자동으로 아이콘 선택
icon_draw_signal_u8g2(u8g2, -60, 107, 6);  // -60dBm, x=107, y=6
```

## 추후 개선 사항
1. 실제 배터리 센서 값 연동
2. 실제 RSSI 값 동적 업데이트
3. 아이콘 애니메이션 효과 추가
4. 추가 상태 아이콘 구현 (예: 블루투스, SD 카드 등)

## 빌드 확인
- 아이콘 라이브러리가 성공적으로 빌드되어 프로젝트에 통합됨
- RX/TX 페이지에서 아이콘이 정상적으로 표시됨
